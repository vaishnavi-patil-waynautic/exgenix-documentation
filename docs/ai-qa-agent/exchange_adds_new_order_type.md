# Usecase:  Exchange Adds New Order Type

---

## AI QA Agent Workflow:

1. **Requirement Sync**  
   - JIRA requirement automatically synced with AI QA agent.

2. **Test Case Generation**  
   - AI generates 20+ test cases covering all possible scenarios.

3. **QA Review**  
   - QA reviews the generated test cases.  
   - 18 test cases accepted as-is, 2 modified for accuracy.

4. **Organization & Traceability**  
   - Test cases organized under the **"Order Management"** module.  
   - Each test case linked back to the JIRA requirement for full traceability.

5. **Execution & Reporting**  
   - Test cases executed via **Exgenix** with full reporting capabilities.
