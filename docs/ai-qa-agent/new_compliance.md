# Usecase:  New Compliance Rule Affects Risk Calculations

---

## AI QA Agent Workflow:

1. **Change Request Creation**  
   - Business analyst creates a change request in **JIRA**.

2. **Test Case Identification**  
   - AI identifies all related test cases (**15 across 3 modules**).

3. **Test Case Suggestions**  
   - AI suggests modifications to existing tests and proposes new test scenarios as needed.

4. **QA Updates**  
   - QA updates test cases with AI assistance, ensuring accuracy and completeness.

5. **Impact Analysis**  
   - Full impact analysis performed, highlighting all affected areas.

6. **Regression Suite Update**  
   - Regression suite updated automatically to reflect changes.

---

## Outcome

- **Complete impact analysis in minutes**, ensuring **no coverage gaps**.
